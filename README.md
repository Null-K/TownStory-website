# TownStory 服务器官网

TownStory Minecraft 服务器的官方网站，使用 VitePress 构建。


## 技术栈

- VitePress
- Vue 3
- TypeScript

## 本地开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run docs:dev
```

### 构建生产版本

```bash
npm run docs:build
```

### 预览生产版本

```bash
npm run docs:preview
```
