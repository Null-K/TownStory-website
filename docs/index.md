# 常见问题

:::details 如何加入服务器？
服务器仅支持正版，请确保使用正版 Minecraft 账号。在多人游戏中直接输入服务器地址 `城镇物语.中国` 即可加入。
:::

:::details 服务器支持哪些版本？
服务器目前支持 `Java 1.26.1` 版本。建议使用核心相同版本以获得最佳体验。
:::

:::details 服务器有什么规则？
我们倡导共建、社交与尊重。禁止作弊、恶意破坏、骚扰其他玩家等行为。详细规则请查看服务器文档或联系管理员。
:::

:::details 遇到问题如何反馈？
你可以加入我们的官方群 `1046069306` 进行反馈或者询问。在游戏中遇到问题或需要帮助时，也可以在游戏内使用聊天功能联系在线管理员。
:::

:::details 服务器会定期备份吗？
服务器会每天自动异地备份数据，并保留最近 7 天的备份记录，以确保数据安全。
:::

:::details 服务器会定期维护吗？
服务器会在每日凌晨 5 点进行重启，每周四进行例行维护。其余维护时间通常会在官方群内提前通知。服务器团队拥有多年开发经验，致力于为玩家提供长期稳定的游戏体验。
:::
